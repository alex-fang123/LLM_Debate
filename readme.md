这是一个利用AutoGen写的多Agent讨论的脚本。

有analyst、advocate、critic和manager四个agent，从效果看，肯定是没办法直接拿来用的。

就是，从样例结果看，用的deepseek v3.2，能够比较出简单的事实，但并不深入，当然这个问题可以通过人的不断引导来达到目的，脚本也支持过一段时间人手动去介入并提出引导——但这样的话就不是一个全自动化的东西。

更大的问题我认为是，有点过于注重逻辑推演——也许这个问题源于Agent并没有太多工具可用，只能从网上收集信息+嘴炮。

或者从形态上说，这就是简陋的模型+讨论，讨论的价值就是能从不同角度不同思路去探索，类似gemini deep think的青春迷你版，但光有思路是远远不够的。
